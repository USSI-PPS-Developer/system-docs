## 1. Pendahuluan
**Digit By IBS** adalah aplikasi mobile untuk **mendaftar sebagai nasabah**, **mengaktivasi
akun**, **membuka rekening** (tabungan/deposito/kredit), serta **memantau saldo, mutasi, dan
analisa finansial** Anda. Pendaftaran & pengajuan Anda akan **diverifikasi oleh petugas BPR**
sebelum diproses; aplikasi menampilkan status prosesnya.

## 2. Persyaratan Penggunaan

| Item | Detail |
|------|--------|
| Perangkat | Android atau iOS (kamera aktif) |
| Aplikasi | **Digit By IBS** (dari tautan resmi / toko aplikasi) |
| Koneksi | Internet aktif |
| Dokumen | **KTP** asli untuk difoto; data diri lengkap |

## 3. Registrasi (Calon Nasabah Baru)
1. Buka aplikasi → di halaman **Login**, pilih **Daftar / Registrasi**.
2. **Foto KTP**: posisikan KTP dalam bingkai, ambil foto. Aplikasi akan **membaca KTP otomatis
   (OCR)** dan mengisi sebagian data.
3. Lengkapi/periksa **data diri**: NIK, nama, tempat & tanggal lahir, jenis kelamin, no HP, email,
   alamat, nama ibu kandung.
4. **Foto selfie**: ikuti panduan kamera depan.
5. Tekan **Kirim**. Data Anda masuk antrean **verifikasi petugas BPR**.

> Pastikan foto KTP **jelas & tidak silau**, dan selfie sesuai wajah pada KTP, agar verifikasi lancar.

## 4. Aktivasi Akun (OTP)
1. Setelah registrasi disetujui, buka menu **Aktivasi**.
2. Tekan **Minta OTP** — kode dikirim/diterbitkan sistem.
3. Masukkan **kode OTP**.
4. Tetapkan **Username** dan **Password** untuk akun Anda.
5. Tekan **Aktivasi**. Bila berhasil, Anda dapat **Login**.

## 5. Login & Keamanan
1. Di halaman **Login**, masukkan **username** & **password**, lalu **Masuk**.
2. Demi keamanan, aplikasi akan **logout otomatis** bila tidak ada aktivitas beberapa menit, dan
   **meminta login ulang** setiap kali aplikasi dibuka kembali.
3. **Ganti password** kapan saja dari menu **Profil**.

## 6. Panduan Fitur

### 6.1 Beranda
Menampilkan **saldo** (geser kartu untuk pindah rekening), **daftar rekening**, **banner/berita**,
dan **menu cepat** ke fitur lain.

### 6.2 Rekening (Saldo & Mutasi)
1. Buka menu **Rekening**.
2. Pilih rekening → lihat **saldo** terkini.
3. Untuk **mutasi**, pilih **rentang tanggal** → daftar transaksi ditampilkan.

### 6.3 Buka Rekening (Tabungan / Deposito / Kredit)
1. Buka menu **Buka Rekening** → pilih jenis: **Tabungan**, **Deposito**, atau **Kredit**.
2. Ikuti **langkah wizard** dan isi data yang diminta:
   - **Tabungan**: produk & setoran awal.
   - **Deposito**: jangka waktu & setoran awal.
   - **Kredit**: tujuan, jangka waktu, nominal pinjaman, penghasilan, pendidikan, tempat/alamat
     kerja, jenis & deskripsi agunan.
3. **Unggah dokumen** sesuai kebutuhan (KTP, selfie, dan untuk kredit: KK, NPWP, foto agunan).
4. Periksa ringkasan → tekan **Kirim**.
5. Pantau **status pengajuan** & buka **detail** untuk melihat perkembangan.

### 6.4 Analisa Finansial
Buka menu **Analisa Finansial** → pilih periode → lihat **grafik** ringkasan finansial Anda.

### 6.5 Info Kantor
Buka menu **Info Kantor** untuk melihat **daftar kantor/cabang** BPR.

### 6.6 Profil
Berisi **data diri** Anda, tautan **website** & **kontak** BPR, tombol **Ganti Password**, dan
**Logout** (dengan konfirmasi).

## 7. FAQ (Pertanyaan Umum)

| Pertanyaan | Jawaban |
|------------|---------|
| Kenapa registrasi belum bisa aktivasi? | Registrasi harus **disetujui petugas** dulu sebelum OTP aktivasi tersedia. |
| Kenapa saya tiba-tiba ter-logout? | Sesi berakhir otomatis saat idle & saat aplikasi dibuka ulang, demi keamanan. Login kembali. |
| Foto KTP saya gagal terbaca | Ulangi foto di tempat terang, KTP rata dalam bingkai, tanpa pantulan cahaya. |
| Berapa lama pengajuan rekening diproses? | Bergantung verifikasi petugas BPR; pantau di **status pengajuan**. |
| Data saya aman? | Password diamankan sebelum dikirim & data komunikasi dienkripsi. Jangan bagikan OTP/password Anda. |

## 8. Troubleshooting

| Masalah | Kemungkinan Penyebab | Solusi |
|---------|----------------------|--------|
| Tidak bisa login | Username/password salah atau akun belum aktif | Periksa kredensial; selesaikan aktivasi; hubungi BPR |
| Aplikasi tidak memuat data | Koneksi internet / server bermasalah | Cek koneksi; coba lagi; laporkan bila menetap |
| Kamera tidak terbuka | Izin kamera belum diberikan | Aktifkan izin kamera di pengaturan perangkat |
| OTP tidak diterima | Nomor HP salah / belum diverifikasi | Pastikan no HP benar; minta ulang OTP; hubungi BPR |
| Gagal kirim pengajuan | Dokumen belum lengkap / koneksi terputus | Lengkapi dokumen; pastikan koneksi; ulangi |
